Version 2.1
Option Beschreibung mögliche Werte box3d thrixty
Hauptpfad Sucht automatisch nach den zugehörigen Files.
"size" wird dabei mit in den Pfad eingebaut.
"[http://base.URL][object]/[size]/Filelist.txt" 		[Pfad] data-baseURL
Objektname [String] data-object
Objektgröße small, large data-size
Basepath These are telling the game where to find the Filelists.
"[http://basepath][filelist-path-small.txt]"
"[http://basepath][filelist-path-large.txt]"
 [empty], [mainpath] thrixty-basepath
Subpath small small/Filelist.txt, [subpath] thrixty-filelist-path-small
Subpath large large/Filelist.txt, [subpath] thrixty-filelist-path-large
Zoom Control How to move the zoomed image.
Automatically use the mouseposition or drag the cutout. 		progressive, classic thrixty-zoom-control
Zoom Mode Zoomed in images showing in the same or an extra window. inbox, outbox, none thrixty-zoom-mode
Zoom Pointer How to point to the zoomed in position. minimap, marker, none thrixty-position-indicator
Zoom Outbox Position Where the zoom box should spawn. right, bottom, left, top thrixty-outbox-position
Direction Tells Box3D to animate in specified direction. forward, backward data-direction
Reversion Tells the player to reverse image-order. 0 / false / forward,
1 / true / backward 		thrixty-direction
Framerate How many pictures should be shown per second. 20, [Integer 1-22] data-framerate
Cycle Duration Time in seconds each whole turn takes. 5, [integer > 0] thrixty-cycle-duration
Sensitivity X How sensitive the Player will react horizontally to touch gestures.
The finger needs to move at least X pixel. 		20, [Integer] thrixty-sensitivity-x
Autoplay Automatically starts the animation N times after being loaded, then stops. -1 / on (infinite),
[integer] (finite),
0 / off 		thrixty-autoplay
Autoload Let the player load automatically. Is always disabled on mobile devices. on, off thrixty-autoload
Keyboard Turns keyboard controls on or off. true, false data-keyboard always on
Animation Controls the animation to be played "endless", "once" or "repeat"ed. endless, once, repeat data-animate integrated into autoplay
Zoom Box Position Where the zoom should be placed. right, left data-zoom-position
Zoom Box Height Height of the zoom box in pixel. 450px, [Zahl]px data-zoom-height for those you now
can use CSS
Zoom Box Width Width of the zoom box in pixel. 450px, [Zahl]px data-zoom-width
Zoom Box Background Which color the zoom box has. #000000, [Hex] data-zoom-background
Zoom Box Border What kind of border the Zoom Box should have. 2px solid #cccccc, [CSS] data-zoom-border
Zoom Box Margin Margin of the zoom box. 9, [Integer] data-zoom-margin
Zoom Text Background Background color of the zoom box description. #ffffff, [Hex] data-zoom-text_bgcolor
Zoom Text Color Font color of the zoom box description. #000000, [Hex] data-zoom-text_fontcolor
Zoom Text The Text inside the zoom box description.. [Text] data-zoom-text
Zoom Font Font to use in the zoom font desripction. [String] data-zoom-font
Zoom Font Size Font Size to use in the zoom box description. 12px, [Integer]px data-zoom-font_size
Imagepaths Directly feeds "Filelist.txt"-content into the Player. [Text] data-images wont be implemented
Zoom on/off When "zoom": turn on zoom function and enables the button. true, zoom, false [?] data-controls wont be implemented
Repeat Time Controls the time between repeats. 10, [Integer] data-animate_interval wont be implemented
